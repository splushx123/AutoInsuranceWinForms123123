# AutoInsuranceWinForms

WinForms-приложение на C# по теме курсовой работы: **информационная система страховой компании, специализирующейся на автостраховании**.

## Что сделано
- за основу взята архитектура исходного приложения из шаблона: авторизация, главное окно, модульные формы, `Db.cs`, логирование, отчеты;
- предметная область адаптирована под тему курсовой:
  - клиенты;
  - автомобили;
  - договоры страхования;
  - страховые случаи;
  - выплаты;
  - сотрудники;
  - комиссии;
  - отчеты.

## Как открыть проект
1. Откройте `AutoInsuranceWinForms.sln` в Visual Studio 2022.
2. Проверьте строку подключения в `App.config`.
3. Выполните SQL-скрипт `Database/CreateDatabase.sql` в SQL Server.
4. Соберите проект и запустите его.

## Учетные записи для входа
Заданы в `App.config`:
- `admin@autoin.local`
- `manager@autoin.local`
- `adjuster@autoin.local`

## Структура
- `LoginForm` — вход по e-mail;
- `MainForm` — главное окно;
- `ClientsForm`, `VehiclesForm`, `ContractsForm`, `InsuranceCasesForm`, `PayoutsForm`, `EmployeesForm`, `CommissionsForm` — основные модули;
- `ReportsForm` — отчеты и экспорт CSV;
- `Db.cs` — работа с SQL Server;
- `LogService.cs` — журнал действий.

## Примечание
Приложение ориентировано на схему БД из курсовой работы. В SQL-скрипт включены таблицы и базовые ограничения.
